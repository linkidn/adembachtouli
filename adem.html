HTML5 is the latest version of HTML used for creating and structuring content on the web. It includes new features for improved semantic markup, audio and video support, graphics rendering, and better accessibility.

CSS is a styling language used to control the presentation and layout of web content. It allows developers to separate the visual aspects of a website from its structure , making it easier to maintain and update.

Javascript is a scripting language used to add interactivity and dynamic functionality to web pages. It can run in the browser and on the server-side, making it a flexible language for creating dynamic web applications.
